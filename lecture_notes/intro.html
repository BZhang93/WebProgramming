What This Course Is; Course Goals
  • Work on real-world and fun projects
  • Develop content on the web for various platforms
  • Understand the limits of various techniques
  • Delve into topics in Programming Languages, Security, and Software Engineering
  • Teach you basic skills including using distributed source control, web standards, HTTP, and code readability
  • Build your intellectual curiosity
  • Teamwork and collaboration
  • Use your creativity

Class Format
  • Read my notes including project specifications before the next class
  • My notes will be available online at least two days before the start of the next class
  • 15 - 30 minutes of class will be understanding key points of a topic
  • The remaining time of class will be used for a hands-on lab (and we will discuss your findings)
  • There will be guest lecturers
What You Will Need For This Class
  • You are urged to bring your laptop to class
  • For Mac users: Homebrew and Xcode are strongly recommended
  • Software: Terminal, SSH, Git, MongoDB, Node.js
  • Web browsers: Chrome, Firefox, and Safari (i.e., "pick 3")
  • A good text editor
    • Mac: Sublime Text, TextWrangler (free), Code ($$$$)
    • Windows: Sublime Text, Notepad++ (free), TextPad (free)
    • Unix / Linux: vim, emacs, gedit
  • (Alternatively) VMware Workstation (for Windows and Linux) or VMware Fusion (for Mac OS X)
Teaching Style and Expectations
  1. You are responsible for your own learning.
  2. If you want everything gone over in lecture, then this is not the course for you. More importantly, that's not how things work in real life.
  3. It is your responsibility to ask questions early and to ask for help.
  4. You will learn by doing, by asking questions, by making mistakes, and by working with others.
  5. You will learn how to teach yourself and to explore.
  6. We will cover a lot in little time because the web development stack is so big now.
  7. This will be a dense and fast course.
