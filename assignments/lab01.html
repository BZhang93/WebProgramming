Lab 1: A Forking Lab

Due: ASAP

Lab Objective

  1. Practice using Git and GitHub

Instructions

  1. Create a GitHub account if you do not have one already at https://github.com/.
  2. Read https://help.github.com/articles/set-up-git on setting up Git on your computer.
  3. Read https://help.github.com/articles/fork-a-repo to understand what forking in GitHub is, and how to fork a repository.
  4. Fork the project running-dogs in GitHub (at https://github.com/tuftsdev/running-dogs).
  5. Assuming that you successfully cloned your fork of the running-dogs repo into the current directory in terminal, identify the "Phrase That Pays..." There are a few ways to do this but a hint is to run git log in the repository. You may also look into using git reset or git checkout. More help at http://stackoverflow.com/questions/4114095/git-revert-to-previous-commit-how.
  6. Complete the form at https://docs.google.com/spreadsheet/viewform?formkey=dGxNcEN6eUdrRUEyMUVLWXZRbzh1REE6MQ#gid=0
