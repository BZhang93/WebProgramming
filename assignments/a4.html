Overview

In this assignment, you will complete a working version of Frogger using JavaScript, playable on the web browser.

Game Instructions

(Instructions taken from http://www.20goto10.co.uk/frogger/instructions.html)

If you have never played Frogger before, you can play it online at http://www.playfrogger.org/.

You'll start the game with 5 frogs. The first frog is shown on the sidewalk below the traffic.

The object of the game is to hop as many frogs to safety as you can --and to score the most points along the way!

You maneuver Frogger using the arrow keys UP, DOWN, LEFT, RIGHT.

Frogger starts his perilous journey on the sidewalk, facing 5 lanes of cars and trucks. Try to get Frogger through the lanes of traffic. Traffic travels in alternative directions and at different speeds, so be careful! Frogger will get "splooched" if he touches any part of a vehicle or is run over.

The game ends when no frogs are left.

Game Scoring

Points accumulate as follows:

  • (Basic) Successfully jumping Frogger forward: 10 points
  • (Basic) Successfully jumping Frogger home: 50 points
  • (Basic) Successfully jumping 5 frogs home: 1000 points
  • (Additional) Taking a lady frog home: 200 points
  • (Additional) Eating a fly: 200 points
  • (Additional) Extra points for time left: 10 points per remaining second

Every time you score 10,000 points and have fewer than 4 frogs left, you'll get another Frogger!

Basic Requirements

You will update your game.js file in your frogger directory (from your Assignment 2). Updates that you will need to make include handling the keypresses and animation.

For this assignment, you are allowed to use jQuery.

Programming should be done on your local computer.

You are not allowed to create additional stylesheets or HTML files.

At the very least, you must implement the following:

  1. Ensure that the game loads immediately on page load. That is, no refresh to load the game (unlike what many of you did in Assignment 2).
  2. Move Frogger on the canvas using the UP, DOWN, LEFT, and RIGHT arrow keys on your keyboard.
  3. Animate the vehicles and logs on the screen. For basic requirements, you are not required to have alligators, snakes, or turtles.
  4. Collision detection: if Frogger is hit by a vehicle or hits water, then player loses a life.
  5. Basic game scoring as illustrated above.

Implementing the basic requirements along with updating the README will give you a grade of a "B-" (80%) for this assignment.

Additional Implementations

In order the achieve an "A" for this assignment, you must accomplish four (4) additional items of your choice from the following list:

  1. Implement a timer and the additional game scoring items as illustrated above
  2. Animate Frogger to dead_frog.png if he gets run over by vehicle, eaten by alligator, hits water, or misses home
  3. Sound (i.e., incorporate the game audio)
  4. Implement level progression. That is, level starts at 1. After jumping 5 frogs home, increment level and make level progressively more difficult (e.g., make vehicles faster).
  5. Incorporate alligators, snakes, and turtles into the game. Should Frogger collides with any of the reptiles, the player loses a life.
  6. Incorporate a fly that shows up at random times. Should Frogger eats the fly, player earns 200 extra points.
