Web Programming

Lab 6: Ms. Pac-Man

This lab is worth 3 points.

Lab Objectives

  • Practice writing JavaScript.
  • Practice using the DOM API.
  • Practice using the JavaScript Canvas API

Overview

Originally created by Google as a playable doodle for Pac-Man's 30th anniversary on May 22, 2010, it also apparently cost the economy approximately $120 million in lost productivity (you can still play it at http://www.google.com/doodles/30th-anniversary-of-pac-man).

Getting Started

  1. If you have not done so already, follow the instructions under "Getting Started" in Assignment 2. You will need to use your private GitHub repo for this lab.
  2. It is expected that you work off of the same folder on your computer that is set up to your private repo. You can verify this by viewing the .git/config file and see what the remote URL is!
  3. Assuming that you have set up a local copy of your private repo on your computer, create a new folder inside of it called mspacman.

Instructions

Provided the sprite sheet pacman10-hp-sprite.png (see above), render onto an HTML5 canvas the blue (not gray) game board with one of the Ms. Pac-Man and one of the ghosts on the game board. Copy the above pacman10-hp-sprite.png image into your mspacman folder. All JavaScript must be in the file mspacman.js file that is called from a mspacman.html document. One CSS file mspacman.css is also required to style the canvas.

Viewing Your Work

Open your mspacman.html page on any web browser that supports the HTML5 canvas (e.g., the latest versions of Chrome, Firefox, Safari, or Microsoft Internet Explorer).

Debugging

Be sure to look at your JavaScript console (e.g., in Chrome Developer Tools or in Firebug)

Submitting the Lab

Commit and push your lab to your private repo. The files that must be in your private repo under mspacman are:

  1. mspacman.css
  2. mspacman.html
  3. mspacman.js
  4. pacman10-hp-sprite.png
  5. README.txt or README.md

Assessment

  1. (1 point) Rendered blue game board
  2. (1 point) Rendered one Ms. Pac-Man
  3. (1 point) Rendered one ghost
  4. (-0.5 point) Outputs exist in JavaScript console
  5. (-3 points) You used jQuery
