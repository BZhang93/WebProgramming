Web Programming

Lab 7: Friends of Ming Chow Jobs Listing

This lab is worth 3 points.

Lab Objectives

  • Practice using JavaScript associative arrays.
  • Practice parsing JSON data.

Getting Started

Instructions

Viewing Your Work

Open your jobs.html page on any web browser.

Debugging

Be sure to look at your JavaScript console (e.g., in Chrome Developer Tools or in Firebug)

Submitting the Lab

Commit and push your lab to your private repo. The files that must be in your private repo under jobs are:

  1. jobs.html
  2. jobs.css
  3. README.txt or README.md

Assessment

  1. (1.5 points) Parsed JSON correctly
  2. (1.5 point) Rendered all jobs on the page
  3. (-0.5 point) Outputs exist in JavaScript console
  4. (-3 points) You used jQuery
