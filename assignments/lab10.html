Web Programming

Lab 10: Getting Started with Node.js on Heroku

This lab is worth 1 point

Lab Objectives

  • Get started with Heroku, Node.js, and the Express web framework

Instructions and Submission

  1. Install Node.js for your computer: http://nodejs.org/. If you have a Mac, I strongly recommend using Homebrew instead of downloading the latest version of Node.js from the Node.js website --see http://mxcl.github.com/homebrew/
  2. Create a Heroku account: http://www.heroku.com/
  3. RTFM (up to "Advanced HTTP features"): https://devcenter.heroku.com/articles/nodejs
  4. Complete the form at https://docs.google.com/forms/d/17_jRmQ_K_aEEzDe_wwRvMZ2ITxWS5WRZPNqIdtDsMgc/viewform
