Web Programming

Lab 6: Friends of Ming Chow Jobs Listing

This lab is worth 3 points.

Lab Objectives

  • Practice using JavaScript associative arrays.
  • Practice parsing JSON data.

Overview

It's all about who you know --and who they know --and who knows you. (read)

Getting Started

  • In your private repo comp20-username on your computer, create a new folder inside of it called jobs.
  • Copy-and-paste the HTML of this page (or right-click and save) to an HTML file named jobs.html in your jobs folder.

Instructions

Parse the string to JSON and render all the jobs on the page in the <div id="list"> section of the body. For each job, show only the following fields:

  1. Company
  2. Location
  3. Position

For this lab, I will encourage creativity: you are free to display all the jobs on the page however you want it to look. That is, you can display all the jobs in a table or do something more simple or more elaborate. You are encouraged to use stylesheet(s).

However, you are NOT ALLOWED to add additional HTML inside of the <body>...</body>! You are also not allowed to use jQuery!

Example

Viewing Your Work

Open your jobs.html page on any web browser.

Debugging

Be sure to look at your JavaScript console (e.g., in Chrome Developer Tools or in Firebug).

Submitting the Lab

Commit and push your files to your private repo on GitHub. You must also include any additional files such as stylesheets.

Assessment

  1. (1.5 points) Parsed JSON correctly
  2. (1.5 point) Rendered all jobs on the page
  3. (-1.0 point) Outputs exist in JavaScript console
  4. (-3.0 point) You added additional HTML tags inside the body!
  5. (-3.0 point) Errors exist in the JavaScript console
  6. (-3.0 point) You used jQuery

Hints and Notes

  • You only need to modify the parse() function, make modifications starting at the // Start working here... mark.
  • You may be asking yourself: isn't there a better way to do this? That is, shouldn't the data be pulled from a web server? Yes, of course --that is where AJAX is necessary, and the topic of next class.
