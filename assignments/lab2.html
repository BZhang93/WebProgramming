Web Programming

Lab 2: Using Chrome Developer Tools

Lab Objectives

  1. Practice using Chrome Developer Tools
  2. Understand how HTTP communications work
  3. Dissect HTTP responses and request headers

Overview

Chrome Developer Tools "allows web developers and programmers deep access into the internals of the browser and their web application." Throughout this course, you will be using Chrome Developer Tools heavily, including debugging and finding bugs, inspecting assets and resources for a web page or web application, and measuring performance. In this lab, you will practice using Chrome Developer Tools to understand how HTTP works.

Please note that most of the functions of Chrome Developer Tools are also available in Firebug for the Firefox web browser. Firebug is an add-on that you must install for Firefox: https://addons.mozilla.org/en-us/firefox/addon/firebug/.
