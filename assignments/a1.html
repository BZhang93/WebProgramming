Web Programming

Assignment 1: Your Portfolio

Due: Friday, February 1st at 5 PM

This assignment is worth 10 points. You must work on this assignment individually.

Overview

While this is a rather simple assignment, it serves two important purposes:

  1. Use HTML5 and CSS.
  2. Help you put together a personal portfolio: whether you are looking for a job after graduation or continue with graduate school.

Instructions

Using only HTML5 and CSS, you are to create three (3) personal web pages:

resume.html - Your resume. It shall include the following sections:

  1. Full name. IMPORTANT: For privacy reasons, please do not list your addresses or telephone numbers.
  2. Education. Do not list high school as it is not important.
  3. Relevant Coursework
  4. Professional Experience. For each job, list accomplishments. Use the past-tense if you no longer work at the job; in the present-tense if you are currently employed at the job. Use bullets-points, an unordered list.
  5. Skills (e.g., programming languages)
  6. Activities that are related to your field of study or work

bio.html - A short biography of yourself.

index.html - Your personal home page. This page is accessed by http://YOUR_GITHUB_USERNAME.github.com (notice the index.html is not necessary at the end of the URL. It is inferred to be index.html). This page shall contain:

  1. Your name
  2. A photo of you (real or fictitious)
  3. A link to your resume
  4. A link to your short biography
  5. A link to your Lab 3

Guidelines

You must adhere to the following guidelines:

  1. Your must use at least two (2) CSS files for styling (e.g., one for desktop and one for mobile). Both of the CSS files must have at least five (5) rules.
  2. All pages must pass the official HTML5 specifications via http://validator.w3.org/
  3. All CSS files must pass the official CSS validation via http://jigsaw.w3.org/css-validator/
  4. You cannot use frames (your page will not pass HTML5 validation anyway).
  5. You cannot use any styling element for any purpose (e.g., <strong>, <em>, <font>, <center>). All styling must be defined in CSS.
  6. You cannot use any styling attribute in elements (e.g., <h3 align="center">).
  7. You cannot use absolute URLs.
  8. You cannot use front-end frameworks such as Bootstrap.
  9. You cannot use JavaScript (or jQuery for that matter; considering we have not covered JavaScript yet!).
  10. All file names must be in lowercase and only [a-z0-9_.] characters can be used for file names (universal standard). That is, do not use whitespace in file names!
  11. All content must be stored in your special repo YOUR_GITHUB_USERNAME.github.com, master branch, of your GitHub account.

Assessment

  1. (0.5 point) All files including directories are in you special repo YOUR_GITHUB_USERNAME.github.com of your GitHub account.
  2. (1 point) Used git constantly: committing with meaningful messages and pushing to GitHub.
  3. (1 point) A README (e.g., README.txt) file that briefly describes your work.
  4. (2 points) Use two (2) CSS files; both having at least 5 rules.
  5. (1 point) All pages pass the official HTML5 specifications; warnings acceptable.
  6. (1 point) All CSS files pass the official CSS validation; warnings acceptable.
  7. (3 points) The three (3) pages with exact file name: index.html, resume.html, and bio.html
  8. (0.5 point) Your index.html contains a link to your Lab 3.
