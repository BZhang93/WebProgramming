Lab Objectives

  • Practice writing JavaScript.

Getting Started

  1. If you have not done so already, follow the instructions under "Getting Started" in Assignment 2. You will need to use your private GitHub repo for this lab.
  2. Assuming that you have set up a local copy of your private repo on your computer, create a new folder inside of it called status_dom.
  3. Copy all the HTML of this page, and paste into an HTML file named profile.html in your status_dom folder.

Instructions

Modify the page, your copy of profile.html, so that messages entered (that is, on pressing the enter / return key) will be displayed dynamically (i.e., no page refresh) in the "Updates" section --with the latest update showing first! All displayed messages must be preceded with a timestamp in the format MM/DD/YYYY HH:MM. A space followed by a hyphen followed by a space must follow the timestamp. Thus, the format of a message to be displayed must be:

MM/DD/YYYY HH:MM - the message

The time must be in 24-hour format! You do not need to display the leading zero for either the date or for the time (example: "9" is acceptable for "09").

Hints and Notes

  1. You will need to add a <script> section in the <head> of the page.
  2. You will need to add and use the onchange attribute for the text input field.
  3. To extract the text inside of the text input field in JavaScript, there is a value attribute for a DOM element. That is, document.getElementById("msg").value
  4. You will need to use the JavaScript Date object to get every piece of the date and time. Please refer to http://www.w3schools.com/jsref/jsref_obj_date.asp for methods of the object.
  5. To learn more about the getElementById() method, please refer to http://www.w3schools.com/jsref/met_doc_getelementbyid.asp.
  6. You will need to work with DOM nodes, including creating new element nodes. For a list of DOM node methods, please refer to http://www.w3schools.com/jsref/dom_obj_node.asp. A couple of methods are vital to the completion of this lab.
