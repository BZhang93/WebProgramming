Overview

The web is an integral part of society and our lives. The web browser has also grown to be a critical piece of software on many platforms including mobile devices and video game consoles. This course will follow the course tradition of "looking under the hood," exploring ways to create web content and applications. Core technologies including HTML5, CSS, JavaScript, JSON, PHP, Git, Ajax, and responsive design will be emphasized. Web security will also be discussed heavily. Finally, this course will delve into modern web development with APIs and JavaScript libraries such as jQuery, and designing for mobile and tablets.

Frequently Asked Questions

Q: What is the web browser of choice for this class?

A: All modern web browsers will be used in this class. However, we will use Chrome and Firefox (with Firebug plugin) extensively.

Q: Should I bring a laptop to class?

A: Yes, to follow demonstrations and to do lab exercises.

Q: What should I use to do the labs and assignments in this class?

A: An editor which you can download, open, edit, and save remote files is ideal. For Windows, use Notepad++ (free). For Mac, use Coda ($99) or TextWrangler (free)

Q: How do we submit assignments?

A: You will need to push your work to a private GitHub repository which I will create for each student.

Q: I am thinking of becoming an entrepreneur. Can this course be a springboard for my tech venture?

A: Sigh. Potentially, yes. GetchaBooks was developed in the winter of 2009 after I taught the class. But don't get your hopes up too high. Please note: you cannot use the department's resources to run your business!

Q: What is the value of this course to the Computer Science curriculum?

A:

  1. This is one of the few courses in the curriculum that requires the use of revision control.
  2. This is one of the few courses in the curriculum where reading and using other people's code (read: APIs) is important.
  3. This is one of the few courses in the curriculum that actually delves into Security, a topic that I and the Accreditation Board for Engineering and Technology (ABET) looks very highly on.
  4. This is one of the few courses in the curriculum that covers the HTTP protocol, networking, data persistence (e.g., cookies, sessions, relational database, non-relational databases, local storage), event-driven programming.
  5. This is one of the few courses in the curriculum where students work in teams.

Perhaps read this by a colleague of mine. I didn't even mention the course's rather obvious values to the students (provides opportunities and the tangible skills that will last a lifetime).
